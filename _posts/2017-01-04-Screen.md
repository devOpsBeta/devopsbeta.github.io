date: "2017-01-04"

---


```
# create session
screen 

# create named session
screen -S <sessName>

# list screen sections
screen -ls

# reconnect to a unnamed session
screen -r <enoughSessionIDNumberToBeUnique>

# reconnect to named session
screen -xS <sessName>

# to disconnect
ctrl+a then d 

# to scroll output of screen session
ctrl+a then esc 





```
